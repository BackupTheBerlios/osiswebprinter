Zugriffsrechte

 

Diese Anleitung soll Ihnen helfen die Zugriffsrechte für die OSIS - Web Printer Konfigurations-Dateien auf einem Windows - System einzurichten.

 

 

  
     
 

Gehen Sie mit Ihrem Windeos Explorer das Verzeichnis

~/includes

 

  

 

Klicken Sie mit der RECHTEN Maustaste auf die Datei config-old.php

 

 

  
     
  Und wählen in dem Menue per klick Eigenschaften aus.  

 

Zugriffsrechte

Für die OSIS WebPrinter Installationsrutine unter http://[Ihre Seite]/html/install/ benötigen Sie Schreibrechte für die Dateien:

~/includes/config-old.php

und

~/includes/config.php

 

 

  
     
 

Klicken Siue auf Übernehmen

  

 

Shcreibschutz

Nach erfolgreicher Installation verwenden Sie für folgende Daten einen Schreibschutz

~/includes/config-old.php

und

~/includes/config.php

 

 

  
     
 

Klicken Sie auf Übernehmen
