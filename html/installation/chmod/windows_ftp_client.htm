Zugriffsrechte

 

Diese Anleitung soll Ihnen helfen die Zugriffsrechte für die OSIS - Web Printer Konfigurations-Dateien per Windows FTP Client auf einem Unix System einzurichten.

In dieser Anleitung verwenden wir den FTP-Cleint WS_FTP

 

  
     
 

Melden Sie sich auf Ihrem Server an und gehen in das Verzeichnis

~/includes

 

  

 

Klicken Sie mit der RECHTEN Maustaste auf die Datei config-old.php

 

 

  
     
  Und wählen in dem Menue per klick chmod aus.  

 

Chmod 666

Für die OSIS WebPrinter Installationsrutine unter http://[Ihre Seite]/html/install/ benötigen Sie Schreibrechte für die Dateien:

~/includes/config-old.php

und

~/includes/config.php

 

 

  
     
 

Klicken Siue auf OK

  

 

Chmod 444

Nach erfolgreicher Installation verwenden Sie für folgende Daten den Chmod 444

~/includes/config-old.php

und

~/includes/config.php

 

 

  
     
 

Klicken Sie auf OK
